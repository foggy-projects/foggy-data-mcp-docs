## 版本规则

- 白皮书版本是公开文档和语义契约版本，不等同于 Java 引擎、Python 引擎或 Odoo 集成包的发布版本。
- 新能力进入后续白皮书版本或单独能力文档，不追加到已经冻结的版本中。
- v2.0 是 v1.0 之后的新增能力续篇，不宣布 v1.0 废弃，也不替代 v1.0 的基础语法参考。
- Java 和 Python 实现可以保留各自包版本号；后续应通过兼容性矩阵说明它们对应的公开契约版本。
