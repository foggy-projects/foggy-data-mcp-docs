# 控制流

## if 语句

```javascript
if (condition) {
    // code
} else if (condition2) {
    // code
} else {
    // code
}
```

## switch 语句

```javascript
switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // default code
}
```

## for 循环

```javascript
// C 风格 for
for (var i = 0; i < 10; i++) {
    // code
}

// for-in (遍历对象键)
for (let key in obj) {
    // code
}

// for-of (遍历数组值)
for (const value of array) {
    // code
}
```

## while 循环

```javascript
while (condition) {
    // code
}
```

## 循环控制

```javascript
break;          // 退出循环
continue;       // 跳过当前迭代
```

## 三元运算符

```javascript
condition ? valueIfTrue : valueIfFalse
```
